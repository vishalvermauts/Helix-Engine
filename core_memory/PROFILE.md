All Python additions, script updates, or architecture refactors must perfectly match our FastAPI structure using modern async/await syntax, `httpx`, and asynchronous subprocess execution (`asyncio.create_subprocess_exec`).

Every single function signature must include explicit Python type hints and leverage Pydantic models for clean incoming data parsing and validation.
