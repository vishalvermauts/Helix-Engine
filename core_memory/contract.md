The project is a single-page marketing website for a construction company. The design must be a 'dark mode' theme. The color palette will consist of a dark charcoal primary background (#111827), a slightly lighter gray for cards/containers (#1F2937), a white/light-gray for text (#F9FAFB), and a bold, high-contrast accent color (amber/yellow, #F59E0B) for buttons, links, and highlights, fitting the construction theme. Typography will be a clean, modern sans-serif font (e.g., Inter or similar system font). The layout will be responsive, utilizing Flexbox and a single-column layout on mobile, expanding to a wider layout on larger screens. The page structure will include a Header with navigation, a Hero section with a strong call-to-action, a Services section, an About section, and a functional Contact Form. The contact form will capture user's name, email, and message, with client-side validation handled by JavaScript.
