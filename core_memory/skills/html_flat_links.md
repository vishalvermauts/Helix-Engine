# HTML Generation Skill

When generating HTML pages, you must adhere to the following rules:

1. All relative links MUST be flat (e.g. href="index.html" or src="style.css").
2. DO NOT use directory prefixes like `workspace/` or `./`.
3. Ensure proper semantic structure.
4. Ensure responsive design is baked in using standard utility classes.
